12.1

There does not exist any Turing machine HHH that behaves as required by Definition 12.1. The halting problem is therefore undecidable.

12.2

If the halting problem were decidable, then every recursively enumerable language would be recursive. Consequently, the halting problem is unde­cidable.

Concept

Computability and Decidability

We say that a problem is decidable if there exists a Turing machine that gives the correct answer for every statement in the domain of the problem.

Halting problem

Given the description of a Turing machine MMM and an input www, does MMM, when started in the initial configuration q0wq_0wq0​w, perform a computation that eventually halts?

Reducing Undecidable Problem

We say that a problem AAA is reduced to a problem BBB if the decidability of AAA follows from the decidability of BBB.

Then, if we know that AAA is undecidable, we can conclude that BBB is also undecidable.

Post Correspondence Problem

Given two sequence of nnn strings on some alphabet Σ\SigmaΣ, say

We break it into two parts.
